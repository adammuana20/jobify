## 💼 Jobify – Job Tracking Platform (MERN)

A fullstack job tracking application that allows users to manage job applications, track statuses, and visualize progress. Built using the MERN stack with secure authentication and a responsive UI.

## 🚀 Live Demo

👉 [https://jobify-jf0q.onrender.com/](https://jobify-jf0q.onrender.com/)

## 📌 Overview

Jobify is designed to help users organize and track their job applications. Users can create, update, and manage job entries while monitoring their application progress through visual dashboards.

⚙️ Installation & Setup
# Clone the repository
git clone [https://github.com/adammuana20/jobify.git](https://github.com/adammuana20/jobify.git)

# Navigate into the project
cd your-repo-name

# Install frontend dependencies
cd client
npm install

# Install backend dependencies
cd ../server
npm install

# Setup environment variables (server)
Create a .env file and add:

MONGO_URI=
JWT_SECRET=

# Run backend
npm run server

# Run frontend
cd ../client
npm run dev
